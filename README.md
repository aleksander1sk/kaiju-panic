# Kaiju Panic

Comedic city-smashing reflex run. You are the kaiju: smash towers into flying rubble, dodge rockets, stomp tanks, roar down drone waves, hurl cars at helicopters — faster every round. Earn scrap, unlock new challenges and skins, take down the MEGA-COPTER boss.

**Play:** https://aleksander1sk.github.io/kaiju-panic/

- HTML5, Phaser 3, Matter physics; all art and sound are procedural (no asset files).
- Portrait 480×800, mouse or touch, one finger is enough.
- Localized: EN, UK, ES, PT, DE, FR, PL, TR.

A game by **OVS-GS** — https://ovs-gs.itch.io

## Dev

```
npm run dev    # local server
npm test       # headless E2E (needs Chrome)
```

This repository is a deploy mirror for GitHub Pages; the canonical source lives in the OVS-GS workspace and is synced by its build pipeline.
